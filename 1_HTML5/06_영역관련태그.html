영역 관련 태그

    영역을 구분 하는 태그
    - <p>   : 단락 구분
    - <pre> : 단락 구분 + 입력한 문자 그대로 표시
    - <div> : 영역을 나누는 태그. (block 형식)
    - <span>: 영역을 나누는 태그. (inline 형식)

div 태그

div 태그 영역은 이미 존재하는 내용의 다음줄에 영역이 설정됨.
-> block 형식 : 공간을 수직 분할하고, width, height 속성이 사용 가능함.

span 태그

span 태그 영역은 이전 태그와의 줄바꿈을 일으키지 않고
바로 옆에 영역이 설정됨.
(inline 형식 : 공간을 수평으로 분할, width, height 속성 사용 X)

div와 span 태그의 영역 지정 방식 차이점

    div 태그 : 사각형의 박스 형태로 영역을 지정

    span 태그 : 내용(content) 단위로 영역을 지정

div 태그 영역 지정 확인

I hear Jerusalem bells are ringing
예루살렘의 종소리와

Roman Cavalry choirs are singing
로마 기병대의 합창이 들리네

Be my mirror my sword and shield
내 거울과 검과 방패가 되어주오

My missionaries in a foreign field
낯선 땅으로 떠나간 내 선교사들이여

For some reason I can't explain
말로 설명할 순 없지만

Once you go there was never
저 높은 곳에 앉으면

Never an honest word
진실함은 존재하지 않네

That was when I ruled the world
내가 통치하던 그때처럼
