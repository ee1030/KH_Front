color : 텍스트의 색상 지정

 
        선택자 {
            color : 색상명 | rgb16진수 | rgb10진수 | rgba | hsl | hsla;
        }
  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb 값으로 지정
  • rgba 값으로 지정
  • hsl 값으로 지정
  • hsla 값으로 지정

text-decoration : 텍스트에 줄을 긋거나 없애는 속성

        선택자 { text-decoration : none | underline | overline | line-through; }
  • 네이버로 이동
  • 밑줄
  • 윗줄
  • 삭제선줄
