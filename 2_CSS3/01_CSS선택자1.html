CSS란?

Cascading Style Sheets의 약자로 마크업 언어가 실제 표시되는 방법을 기술하는 언어.
(웹 페이지 스타일 지정에 사용되는 언어의 표준(W3C에서 표준으로 지정))

CSS 기본 선택자

선택자란? 특정한 HTML 태그를 선택할 때 사용하는 기능으로,
태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다.

모든 요소 선택자 ( * )

선택된 요소 내 모든 태그를 선택할 때 사용

태그 선택자 (태그명)

HTML 문서 내에 같은 태그명을 가진 모든 태그를 선택

 p { color : red; } 와 같은 형태로 사용함.

아이디 선택자 (#)

HTML 문서 내에 해당하는 id 속성 값이 일치하는 태그를 선택

 #id속성값 { color : blue; } 와 같은 형태로 사용.

  2. 아이디 선택자 테스트 2
  3. 아이디 선택자 테스트 3
  4. 아이디 선택자 테스트 4
  5. 아이디 선택자 테스트 5

id 속성의 값은 HTML문서 내에서 유일해야함. (식별자 역할)

클래스 선택자 ( . )

HTML 문서 내에서 동일한 class 속성값을 가진 태그를 선택

 .class 속성값{ color : red; }
