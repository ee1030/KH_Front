반응형 웹(Responsive Web)이란?

PC, 스마트폰, 태블릿, 스마트TV 등 다양한 기기의 디스플레이 크기에 맞게
자동으로 웹 화면의 크기와 구조를 변화하도록 만든 웹 페이지

viewport : 화면상의 표시 영역(디스플레이)

화면 크기가 변하면 페이지 너비나 화면 배율을 조정
* 반응형 웹을 구현하기 위한 가장 기본적인 메타 태그
width: viewport의 가로 크기 조정.
일반적인 숫자값 또는 device-width와 같은 특정 값 사용 가능.
device-width : 웹페이지의 기본 너비를
현재 사용 기기의 화면 너비와 일치
initial-scale : 페이지가 처음 로딩될 때 줌(확대) 레벨 조정(1~10, 기본값 1).
height : viewport의 세로 크기 조정
minimum-scale : viewport의 최소 배율값, 기본값은 0.25입니다.
maximum-scale : viewport의 최대 배율값, 기본값은 1.6입니다.
-scalable : 사용자의 확대/축소 기능을 설정, 기본값은 yes입니다.

미디어 쿼리(@media)

화면(screen), 프린터(print)와 같은 미디어 타입을 따라 다른 스타일 시트를 적용하게 하는 css 코드.
