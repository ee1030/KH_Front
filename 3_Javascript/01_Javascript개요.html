Javascript 개요

스크립트 언어란?

    기존에 이미 존재하는 프로그램의 동작을
    사용자의 요구에 맞게 수행되도록 제어하는 용도의 언어.

    소스코드의 수정이 많은 곳에서 사용하기 적합함.(Interpreter 방식)

    * Interpreter란? 소스코드를 한 줄씩 읽어 바로 실행하는 방식.

    빠르게 배우고 짧은 소스코드 파일로 상호작용을 하도록 고안됨.

Javascript란?

    웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어.
    
    HTML 문서 내에 기술되고, HTML과 함께 웹 브라우저에 의해서 해석이 됨.
    -> 클라이언트 사이드 스크립트 언어

    ECMA 스크립트 표준을 따르는 대표적인 웹 기술.

자바스크립트 작성 방법

    자바스크립트 소스 코드는 함수 단위로 작성하고,
    이벤트 속성을 이용하여 구동되게함.

    이벤트 속성은 태그마다 차이가 있음.
    ->  on이벤트명 = "실행할 함수명([전달값]);"

inline 방식

태그에 직접 간단한 소스코드를 작성해서 실행되게 하는 방법.

마우스 오버 / 마우스 아웃

internal 방식

자바스크립트 내용을 <script> 태그에 작성하여 실행되게 하는 방식.

<script> 태그는 head, body 태그 내부에 작성함.

external 방식

별도의 .js파일을 생성하여 해당 파일에 자바스크립트 코드를 작성하여 가져다 사용하는 방식.
